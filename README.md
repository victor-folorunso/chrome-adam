# Chrome Adam 🚀

Launch all your Chrome profiles instantly with one click.

## What You Need
1. **Chrome Eve** (auto-refresh extension config)
2. **Chrome Adam** (this tool)

---

## Setup Guide

### Step 1: Install Chrome Eve Extension Settings

1. Download **Chrome Eve** from the [Releases](link-to-releases) page
2. In Chrome, click on your **Auto Refresh Plus** extension → Open Settings
3. Scroll to **Side Menu** → **Settings Backup**
4. Click **Import** and select the Chrome Eve file you downloaded

---

### Step 2: Configure Browser Startup Page

1. Open Chrome → Click the **three dots** (⋮) → **Settings**
2. Navigate to **On startup** (left sidebar)
3. Select **Open a specific page or set of pages**
4. Click **Add a new page**
5. Enter: `https://www.fiverr.com/inbox`
6. Click **Add**

---

### Step 3: Run Chrome Adam

Download **Chrome Adam** from [Releases](link-to-releases)

#### Option A: Manual Launch (Simplest)
- Double-click `chrome-adam.exe` whenever you want to open all profiles

#### Option B: Auto-Start on Login (Recommended)
1. Press `Win + R`
2. Type `taskschd.msc` → Press **OK**
3. Click **Create Basic Task**
4. Name it (e.g., "Chrome Adam")
5. Trigger: **When I log on**
6. Action: **Start a program**
7. Program path: Enter full path to exe (e.g., `C:\Users\PC\Desktop\chrome-adam.exe`)
8. Click **Next** → **Finish**
9. Restart your PC to test

#### Option C: Startup Folder (Alternative)
1. Press `Win + R`
2. Type `shell:startup` → Press **OK**
3. Copy `chrome-adam.exe` into the folder that opens
4. Restart your PC
