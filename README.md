## Setup Guide

### Step 1: Install Chrome Eve Extension Settings

1. Download **Chrome Eve** from [Here](https://github.com/victor-folorunso/chrome-adam/blob/main/chrome-eve.arp)
2. In Chrome, click on your **Auto Refresh Plus** extension ‚Üí Open Settings
3. Scroll to **Side Menu** ‚Üí **Settings Backup**
4. Click **Import** and select the Chrome Eve file you downloaded

---

### Step 2: Configure Browser Startup Page

1. Open Chrome ‚Üí Click the **three dots** (‚ãÆ) ‚Üí **Settings**
2. Navigate to **On startup** (left sidebar)
3. Select **Open a specific page or set of pages**
4. Click **Add a new page**
5. Enter: `https://www.fiverr.com/inbox`
6. Click **Add**

---

### Step 3: Run Chrome Adam

Download **Chrome Adam** from [Releases](https://github.com/victor-folorunso/chrome-adam/releases/tag/chrome-adam-v1.0)

#### Option A: Manual Launch (Simplest)
- Double-click `chrome-adam.exe` whenever you want to open all profiles

#### Option B: Auto-Start on Login (Recommended)
1. Press `Win + R`
2. Type `shell:startup` ‚Üí Press **OK**
3. Copy `chrome-adam.exe` into the folder that opens
4. Restart your PC
